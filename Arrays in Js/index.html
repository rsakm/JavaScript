Arrays

Basic Array Methods:

  1. pop() : Deletes an element from last
  2. push() : Adds elements(passed as parameters) at last
  3. shift() : deletes an element from the start
  4. unshift() : Adds elements at the start
  5. concat() : Concates the arrays into a new Array(More than one parameters can be passed
  6. indexOf() : returns the index of an element from the array or -1 if the element is not present
  7. includes() : returns boolean value for the element present in the array or not
  8. reverse() : Reverses the original array
  9. sort() : Sorts the original array Alphabetically in ascending order(Dictionary order)
  10. slice() : creates new array and includes the element at first parameter and excludes the last parameter index
  11. splice() : Creates new array as well as modify the original array
